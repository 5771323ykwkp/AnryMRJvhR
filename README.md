# 校园超市管理系统 java26

## 项目概述

校园超市管理系统是一款基于前后端分离架构的管理系统。该系统集成了供应商管理、商品管理、账单管理等功能，并内置了完善的用户权限管理，适合于校园超市的日常运营管理。

## 技术栈

- **后台框架：**
  - SpringBoot
  - MyBatis-Plus
- **UI界面：**
  - Vue
  - ElementUI
  - Axios
  - Node.js
- **数据库：**
  - MySQL

## 功能模块

- **供应商模块：** 管理供应商信息。
- **超市商品模块：** 管理商品信息及库存。
- **超市账单模块：** 管理账单及销售记录。
- **系统基础模块：**
  - 用户管理
  - 部门管理
  - 角色管理
  - 菜单管理
  - 日志管理
  - 数据字典管理
  - 文件管理
  - 图表展示

## 系统角色

系统基于角色的访问控制，可适用于超市管理员、收银员等不同角色。权限可以精确到按钮级别，支持自定义角色与权限分配。

## 运行环境

- 后端：Java 1.8+，SpringBoot 环境搭建
- 前端：Node.js 环境搭建
- 数据库：MySQL 5.7+
